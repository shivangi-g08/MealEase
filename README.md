# 🍽️ MealEase

**Skip the Queue. Eat Smarter.**

MealEase is a smart canteen pre-ordering system designed for students. It allows users to browse meals, place orders in advance, select pickup slots, and collect food using QR-based verification.

## ✨ Features

- 🍔 Browse food menu
- 🛒 Add items to cart
- ⏰ Select pickup slots
- 📱 QR-based pickup
- 📊 Order status tracking
- 📜 Order history

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- LocalStorage
- QRCode.js
- AOS Animation Library
